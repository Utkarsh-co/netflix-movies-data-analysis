# Netflix Movie Data Analysis (2000-2024)

## 📌 Project Overview
This repository contains an end-to-end data analytics project exploring over 9,600+ Netflix movie records from the years 2000 to 2024. The project demonstrates a full data pipeline workflow: from raw data exploration and aggregation in Microsoft Excel to building a dynamic, interactive business intelligence dashboard in Power BI.

## 🛠️ Tech Stack & Tools Used
* **Microsoft Excel:** Pivot Tables, Data Aggregation, and Exploratory Data Analysis (EDA).
* **Power BI Desktop:** DAX, Data Modeling, New Card Visual Layouts, and Interactive Slicers.

## 🚀 Key Insights Discovered
* **Production Volume:** Movie release frequencies scaled up steadily over two decades, reaching a massive volume peak leading up to 2021 before experiencing a sharp decline.
* **Genre Dominance:** Action, Crime, and Thriller represent the highest overall percentage shares of movies in the dataset. 
* **Dynamic Historical Shifts (2020 Focus):** When filtering specifically for the year 2020, the dashboard highlights unique pandemic-era trends—total releases dropped to 446, "Animation" became the top genre, and *Demon Slayer* emerged as the most popular title by audience metrics.
